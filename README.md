# 🚀 Productivity Dashboard

Современный Dashboard для управления задачами и продуктивностью.

![Preview](assets/screenshot.png)

## ✨ Возможности

* ⏰ Часы в реальном времени
* 📅 Отображение текущей даты
* ✅ Добавление задач
* ❌ Удаление задач
* 💾 Автоматическое сохранение задач
* 📱 Адаптивный дизайн

## 🛠 Технологии

* HTML5
* CSS3
* JavaScript
* LocalStorage

```text
dashboard/
```

## 🚀 Как запустить

### Способ 1

Скачайте проект:

```bash
git clone https://github.com/m1koooo723/dashboard.git
```

Откройте файл:

```text
index.html
```

в любом браузере.

### Способ 2 (VS Code)

1. Откройте папку проекта в VS Code.
2. Установите расширение Live Server.
3. Нажмите правой кнопкой по `index.html`.
4. Выберите `Open with Live Server`.

## 📸 Скриншоты

Добавьте изображение в:

```text
assets/screenshot.png
```
